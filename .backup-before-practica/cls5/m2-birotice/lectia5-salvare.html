De ce trebuie sa salvezi? Scurtatura magica!

Cand lucrezi la un document, modificarile sunt tinute temporar in memoria calculatorului. Daca se inchide calculatorul (sau crapa Word), pierzi tot ce ai lucrat!

Salvarea inseamna sa scrii documentul pe hard disk, unde ramane in siguranta.

Scurtatura magica: Ctrl + S

Apasa Ctrl + S des in timp ce lucrezi! E cea mai importanta scurtatura pe care trebuie s-o retii.

Scurtaturi importante de memorat:

  • Ctrl + S = Save (Salveaza)
  • Ctrl + C = Copy (Copiaza)
  • Ctrl + V = Paste (Lipeste)
Ce scurtatura folosesti pentru salvare rapida?
A Ctrl + C (copiaza)
B Ctrl + S (salveaza)
C Ctrl + V (lipeste)
üí° S vine de la Save (salveaza in engleza). Ctrl + C copiaza, Ctrl + V lipeste.
2

Save vs Save As - Care e diferenta?

Exista doua moduri de a salva un document:

Save (Ctrl + S) Save As (F12)
Salveaza peste fisierul existent Creaza o copie cu alt nume/locatie
Prima data cere nume si locatie Mereu cere nume si locatie

Exercitiu practic:

  1. Deschide Word si scrie: "Test de salvare"
  2. Apasa Ctrl + S
  3. Alege locatia "Desktop" si numele "test-salvare"
  4. Apasa Save
De ce e important sa salvezi des?
A Pentru ca Word iti cere
B Pentru a nu pierde munca daca se inchide calculatorul
C Pentru a face fisierul mai mare
üí° Daca nu salvezi si se inchide programul, pierzi tot ce ai lucrat de la ultima salvare!

Extensii si organizare in foldere

Extensia este terminatia numelui unui fisier, dupa punct. Spune calculatorului ce tip de fisier este.

Extensie Tip fisier
.docx Document Word
.xlsx Tabel Excel
.pdf Document PDF (nu se poate edita usor)
.jpg / .png Imagini

Organizarea in foldere:

Un folder (dosar) este ca un sertar unde pui fisiere similare. Creeaza foldere pentru:

  • Scoala (apoi subfoldere pe materii)
  • Poze
  • Jocuri
Ce extensie are un document Word?
A .jpg (imagine)
B .docx (document Word)
C .mp3 (muzica)
üí° Extensia .docx vine de la DOCument XML - formatul modern al Word.

Crearea si organizarea folderelor

Ca sa gasesti usor fisierele, trebuie sa le organizezi in foldere (dosare).

Cum creezi un folder nou:

  1. Deschide File Explorer (sau mergi pe Desktop)
  2. Click dreapta intr-un loc gol
  3. Alege New (Nou) > Folder
  4. Scrie un nume descriptiv pentru folder

Sugestie de organizare pentru scoala:

  • Scoala (folder principal)
    • Romana
    • Matematica
    • TIC
    • Istorie

Exercitiu practic:

Creeaza pe Desktop un folder "Scoala" cu subfoldere pentru 3 materii!

Cum creezi un folder nou?
A Apesi Ctrl + S
B Click dreapta > New > Folder
C Inchizi calculatorul
üí° Click dreapta in orice loc (Desktop, File Explorer) si alegi New > Folder pentru a crea un dosar nou.
5

AutoSave si AutoRecover - Salvarea automata

Word are functii care te ajuta sa nu pierzi munca, chiar daca uiti sa salvezi!

AutoSave (Salvare automata):

  • Disponibil pentru fisiere salvate in OneDrive sau SharePoint
  • Salveaza automat la fiecare cateva secunde
  • Apare ca un comutator in coltul stanga-sus

AutoRecover (Recuperare automata):

  • Word salveaza o copie de siguranta la fiecare 10 minute
  • Daca Word se inchide neasteptat, iti ofera sa recuperezi documentul

Important:

Chiar daca exista AutoRecover, tot e bine sa apesi Ctrl + S des! E mai sigur!

Ce este AutoSave (Salvare automata)?
A O functie care sterge fisierele automat
B O functie care salveaza documentul automat la cateva secunde
C Un virus de calculator
üí° AutoSave te protejeaza - salveaza automat documentul ca sa nu pierzi nimic daca se inchide Word.
6

Gasirea fisierelor salvate

Uneori uitam unde am salvat un fisier. Uite cum il gasesti:

Metoda 1: File Explorer

  1. Deschide File Explorer (iconita cu folder galben)
  2. Mergi in locatia unde ai salvat (Desktop, Documents, etc.)
  3. Cauta fisierul dupa nume

Metoda 2: Cautare Windows

  1. Apasa tasta Windows sau click pe bara de cautare
  2. Scrie numele fisierului (sau o parte din el)
  3. Windows va gasi toate fisierele care contin acel text

Metoda 3: Recent files in Word

  • Deschide Word > File > Open > Recent
  • Vezi lista cu ultimele documente deschise
Cum gasesti un fisier pe care l-ai salvat?
A Il cauti la gunoi
B Folosesti File Explorer sau bara de cautare Windows
C Trebuie sa-l creezi din nou
üí° File Explorer si bara de cautare din Windows te ajuta sa gasesti orice fisier de pe calculator.
7

Recapitulare: Cele 3 reguli de aur

Hai sa recapitulam cele mai importante lucruri despre salvare!

CELE 3 REGULI DE AUR:

  1. Salveaza DES - Apasa Ctrl + S la fiecare cateva minute. E mai bine sa salvezi de 10 ori decat sa pierzi tot!
  2. Organizeaza in FOLDERE - Creeaza foldere pentru fiecare materie/proiect. Vei gasi fisierele mult mai usor!
  3. Nume CLARE - Numeste fisierele clar: "Referat_Romana_Toamna.docx" nu "Document1.docx"

Sfaturi pentru nume de fisiere:

  • Foloseste underscore (_) sau cratima (-) in loc de spatii
  • Include data daca e important: "Tema_TIC_20ianuarie.docx"
  • Evita caractere speciale: ? * : " < > |
Care este regula de aur pentru salvare?
A Salvezi doar la sfarsit
B Salvezi des, cu Ctrl + S, in foldere organizate
C Nu salvezi niciodata
üí° Regula de aur: Ctrl + S des, foldere organizate, nume clare pentru fisiere!
