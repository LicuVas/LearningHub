Ordinea operatiilor:

  1. ( ) - Paranteze (prioritate maxima)
  2. * / % // - Inmultire, impartire, rest
  3. + - - Adunare, scadere
  4. < > <= >= == != - Comparatie
  5. not / ! - Negatie logica
  6. and / && - SI logic
  7. or / || - SAU logic
üìö EXERCITII REZOLVATE
üìù Exercitiu 1 - Tip Subiect I

Fie a = 10, b = 3. Care este valoarea expresiei:

a % b + a // b * 2

Rezolvare pas cu pas:

  1. a % b = 10 % 3 = 1 (restul impartirii)
  2. a // b = 10 // 3 = 3 (impartire intreaga)
  3. 3 * 2 = 6
  4. 1 + 6 = 7

Raspuns: 7

üìù Exercitiu 2 - Expresie logica

Fie x = 5, y = 10. Care este valoarea expresiei:

x < y and y % x == 0

Rezolvare pas cu pas:

  1. x < y ‚Üí 5 < 10 = True
  2. y % x = 10 % 5 = 0
  3. 0 == 0 = True
  4. True and True = True

Raspuns: True

üìù Exercitiu 3 - Combinat

Fie n = 17. Care este valoarea expresiei:

n // 10 + n % 10 * 10

Rezolvare pas cu pas:

  1. n // 10 = 17 // 10 = 1 (cifra zecilor)
  2. n % 10 = 17 % 10 = 7 (cifra unitatilor)
  3. 7 * 10 = 70
  4. 1 + 70 = 71

Observatie: Aceasta expresie inverseaza un numar de 2 cifre!

Raspuns: 71

Trucuri pentru Bac
  • n % 10 = ultima cifra
  • n // 10 = numarul fara ultima cifra
  • n % 100 = ultimele 2 cifre
  • n // 100 = numarul fara ultimele 2 cifre
  • n % 2 == 0 = numarul e par
  • n % 2 == 1 = numarul e impar
