📚 INVATA

Ce este structura liniara?

Structura liniara (secventiala)
Un program cu structura liniara executa instructiunile una dupa alta, in ordinea in care sunt scrise. Ca o reteta de gatit - faci pasii in ordine, fara sa sari!
Instructiune 1
↓
Instructiune 2
↓
Instructiune 3
↓
...
Sablonul clasic: Citire - Prelucrare - Afisare
// 1. CITIRE - luam datele de intrare cin >> a; cin >> b; // 2. PRELUCRARE - facem calculele suma = a + b; produs = a * b; // 3. AFISARE - aratam rezultatele cout << "Suma: " << suma << endl; cout << "Produs: " << produs << endl;
Exemplu complet: Conversia temperaturii
#include <iostream> using namespace std; int main() { float celsius, fahrenheit; // Citire cout << "Temperatura in Celsius: "; cin >> celsius; // Prelucrare (formula: F = C * 9/5 + 32) fahrenheit = celsius * 9 / 5 + 32; // Afisare cout << celsius << "C = " << fahrenheit << "F" << endl; return 0; }
💡 Probleme tipice cu structura liniara
  • Calcul arie/perimetru figuri geometrice
  • Conversii unitati (km-m, C-F, ore-minute)
  • Calcule matematice simple
  • Interschimbarea valorilor a doua variabile

P1. Suma a doua numere Usor

Citeste doua numere intregi a si b de la tastatura si afiseaza suma lor.

Exemple:
Input 3 5
Output 8
Input 100 250
Output 350
Declara variabile int a, b, suma;, citeste cu cin >> a >> b;, calculeaza suma = a + b; si afiseaza cu cout << suma;

P2. Perimetrul dreptunghiului Usor

Se dau lungimea L si latimea l unui dreptunghi. Calculeaza si afiseaza perimetrul.

Exemple:
Input 5 3
Output 16
Input 10 4
Output 28
Formula perimetrului: P = 2 * (L + l) sau P = 2*L + 2*l

P3. Interschimbare variabile Mediu

Citeste doua numere a si b. Interschimba valorile lor si afiseaza-le (prima valoarea lui b, apoi valoarea lui a).

Exemple:
Input 3 7
Output 7 3
Ai nevoie de o variabila temporara temp: temp = a; a = b; b = temp;

P4. Media aritmetica Usor

Citeste 3 numere intregi si calculeaza media lor aritmetica (cu zecimale).

Exemple:
Input 10 20 30
Output 20
Input 7 8 9
Output 8
Foloseste float pentru medie: media = (a + b + c) / 3.0; (observa 3.0, nu 3)
