Aplicatii de Fizica in C++

Transformam formulele de fizica in programe functionale!

🚀 Miscare rectilinie uniforma

Formula:

d = v * t

d = distanta, v = viteza, t = timp

#include <iostream> using namespace std; int main() { float viteza, timp, distanta; cout << "Viteza (m/s): "; cin >> viteza; cout << "Timpul (s): "; cin >> timp; distanta = viteza * timp; cout << "Distanta parcursa: " << distanta << " metri" << endl; return 0; }
🎯 Miscare rectilinie uniform variata

Formula:

d = v0 * t + (a * t2) / 2

v0 = viteza initiala, a = acceleratie

float v0, a, t, d; cout << "Viteza initiala (m/s): "; cin >> v0; cout << "Acceleratia (m/s^2): "; cin >> a; cout << "Timpul (s): "; cin >> t; d = v0 * t + (a * t * t) / 2; cout << "Distanta: " << d << " m" << endl;
💡 Energie cinetica

Formula:

Ec = (m * v2) / 2

m = masa (kg), v = viteza (m/s)

float masa, viteza, energie; cout << "Masa (kg): "; cin >> masa; cout << "Viteza (m/s): "; cin >> viteza; energie = (masa * viteza * viteza) / 2; cout << "Energia cinetica: " << energie << " Jouli" << endl;

P1. Calculeaza viteza Usor

Citeste distanta d (metri) si timpul t (secunde), apoi calculeaza si afiseaza viteza folosind formula v = d / t.

Exemple:
Input 100 20
Output Viteza: 5 m/s
Input 150 10
Output Viteza: 15 m/s
Formula: viteza = distanta / timp; - Foloseste tipul float pentru precizie!

P2. Celsius la Fahrenheit Usor

Citeste temperatura in grade Celsius c si transforma in Fahrenheit folosind formula F = C * 9/5 + 32.

Exemple:
Input 0
Output 32 F
Input 100
Output 212 F
fahrenheit = celsius * 9.0 / 5.0 + 32; - Foloseste 9.0/5.0 pentru rezultat zecimal!
🚀 OnlineGDB
P3. Energia cinetica Mediu

Citeste masa m (kg) si viteza v (m/s), apoi calculeaza energia cinetica folosind formula Ec = (m * v2) / 2.

Exemple:
Input 10 5
Output Ec = 125 J
Input 2 10
Output Ec = 100 J
energie = (masa * viteza * viteza) / 2; - viteza^2 se scrie ca viteza * viteza
🚀 OnlineGDB
P4. Distanta in cadere libera Mediu

Citeste timpul t (secunde) si calculeaza distanta parcursa in cadere libera folosind d = (g * t2) / 2, unde g = 9.8 m/s2.

Exemple:
Input 2
Output Distanta: 19.6 m
Input 5
Output Distanta: 122.5 m
Declara const float g = 9.8; apoi distanta = (g * t * t) / 2;

🚀 Mini-Proiect: Convertor de unitati fizice Proiect

Creeaza un program care converteste intre diferite unitati de masura fizice. Urmeaza pasii:

1 Afiseaza un meniu: 1-km in m, 2-kg in g, 3-ore in secunde, 4-C in F
2 Citeste optiunea utilizatorului (1-4)
3 Citeste valoarea de convertit si afiseaza rezultatul corect
4 Bonus: Adauga conversie inversa (m in km, g in kg, etc.)
Foloseste if-else sau switch: km->m: *1000, kg->g: *1000, ore->sec: *3600, C->F: *9/5+32
