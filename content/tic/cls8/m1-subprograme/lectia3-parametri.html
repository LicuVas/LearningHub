Ce sunt parametrii?

Parametrii sunt variabilele prin care trimitem date catre functie. Cand declari functia - sunt parametri formali (placeholder-e) Cand apelezi functia - valorile trimise se numesc argumente

Ce sunt parametrii unei functii?
A Rezultatele pe care le returneaza functia
B Variabilele prin care trimitem date catre functie
C Numele functiei
💡 Recitește secțiunea pentru a găsi răspunsul.
2

Transmitere prin valoare

In C++, cand trimiti un parametru, functia primeste o copie a valorii. Modificarile din functie NU afecteaza variabila originala!

Ce afiseaza codul urmator? int test(int a, int b) { return a - b; } cout << test(10, 3);
A -7
B 7
C 13
💡 Recitește secțiunea pentru a găsi răspunsul.
3

Parametri multipli

Poti avea oricate parametri ai nevoie, separati prin virgula. Fiecare parametru trebuie sa aiba tipul specificat!

Cate argumente trebuie sa trimiti cand apelezi: void f(int x, int y, int z)?
A 1
B 2
C 3
💡 Recitește secțiunea pentru a găsi răspunsul.
4

Atentie la ordine!

Cand apelezi functia, ordinea argumentelor CONTEAZA!

Pentru void scrie(string nume, int varsta), care apel este corect?
A scrie(15, "Ana")
B scrie("Ana", 15)
C scrie(Ana, 15)
💡 Ordinea argumentelor trebuie sa corespunda cu ordinea parametrilor.
