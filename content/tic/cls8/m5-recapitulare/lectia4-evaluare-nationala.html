TRY - Probleme Model Rezolvate

Problema 1: Suma Cifrelor

Usor
Cerinta: Scrieti un algoritm care citeste un numar natural n si afiseaza suma cifrelor sale.

Exemplu: Pentru n = 1234, se afiseaza 10 (1+2+3+4=10) 
# Citim numarul
n = int(input("n = "))

# Initializam suma
suma = 0

# Cat timp mai avem cifre
while n > 0:
    cifra = n % 10      # Extragem ultima cifra
    suma = suma + cifra  # Adaugam la suma
    n = n // 10         # Eliminam ultima cifra

# Afisam rezultatul
print("Suma cifrelor:", suma)

Explicatie: Extragem fiecare cifra cu % 10 (restul impartirii la 10), o adaugam la suma, apoi eliminam cifra cu // 10 (impartire intreaga).

Problema 2: Numar Prim

Mediu
Cerinta: Scrieti un algoritm care verifica daca un numar citit este prim.

Exemplu: Pentru n = 17, se afiseaza "Este prim" 
# Citim numarul
n = int(input("n = "))

# Verificam cazurile speciale
if n < 2:
    print("Nu este prim")
else:
    este_prim = True

    # Verificam divizorii de la 2 la sqrt(n)
    for d in range(2, int(n**0.5) + 1):
        if n % d == 0:
            este_prim = False
            break

    if este_prim:
        print("Este prim")
    else:
        print("Nu este prim")

Explicatie: Un numar este prim daca nu are alti divizori in afara de 1 si el insusi. Verificam doar pana la radacina patrata pentru eficienta.

Problema 3: Interogare SQL

Mediu
Context: Se da tabelul PRODUSE cu campurile: ID_Produs, Nume, Categorie, Pret, Stoc.

Cerinte:
a) Afisati produsele din categoria "Electronice" ordonate descrescator dupa pret.
b) Calculati numarul de produse cu stoc mai mare de 100.
c) Afisati pretul mediu pentru fiecare categorie. 
-- a) Produse electronice ordonate dupa pret
SELECT * FROM PRODUSE
WHERE Categorie = 'Electronice'
ORDER BY Pret DESC;

-- b) Numarul produselor cu stoc > 100
SELECT COUNT(*) as Nr_Produse
FROM PRODUSE
WHERE Stoc > 100;

-- c) Pretul mediu pe categorii
SELECT Categorie, AVG(Pret) as Pret_Mediu
FROM PRODUSE
GROUP BY Categorie;

Problema 4: Cifra Maxima

Dificil
Cerinta: Scrieti un algoritm care citeste un numar natural n si afiseaza cifra maxima si de cate ori apare aceasta in numar.

Exemplu: Pentru n = 3539353, cifra maxima este 9 si apare de 1 data. 
# Citim numarul
n = int(input("n = "))
copie = n  # Pastram copia pentru a doua parcurgere

# Gasim cifra maxima
maxim = 0
while n > 0:
    cifra = n % 10
    if cifra > maxim:
        maxim = cifra
    n = n // 10

# Numaram aparitiile cifrei maxime
contor = 0
while copie > 0:
    cifra = copie % 10
    if cifra == maxim:
        contor = contor + 1
    copie = copie // 10

print(f"Cifra maxima: {maxim}")
print(f"Apare de {contor} ori")
