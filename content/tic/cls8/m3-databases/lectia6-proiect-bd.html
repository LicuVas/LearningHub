GHID PROIECT

Pas cu pas

1Creeaza baza de date
  1. Deschide Microsoft Access
  2. Click pe "Blank database"
  3. Numele: ScoalaDB
  4. Locatie: Desktop
  5. Click Create
2Creeaza tabelul Elevi

In Design View, adauga aceste campuri:

Field Name Data Type Observatii
ID AutoNumber Cheie primara
Nume Short Text Max 50 caractere
Prenume Short Text Max 50 caractere
DataNasterii Date/Time Format: Short Date
Clasa Short Text Ex: 8A, 8B
Media Number Decimal Places: 2
3Adauga date de test

Treci in Datasheet View si adauga cel putin 10 elevi. Exemplu:

NumePrenumeDataNasteriiClasaMedia
PopescuMaria15/03/20118A9.50
IonescuAndrei22/07/20118A8.75
PopaElena08/11/20108B9.25
DumitrescuMihai30/01/20118B7.80
StanescuAna14/05/20118A9.00

Adauga inca 5 elevi cu date inventate!

4Creeaza interogarile

Interogare 1: Elevii din clasa 8A

SELECT *
FROM Elevi
WHERE Clasa = '8A';

Interogare 2: Elevi cu media > 8, sortati descrescator

SELECT *
FROM Elevi
WHERE Media > 8
ORDER BY Media DESC;

Interogare 3: Nume si Media pentru elevii din 8B

SELECT Nume, Media
FROM Elevi
WHERE Clasa = '8B';
BONUS: Pentru nota maxima
  • Adauga un al doilea tabel "Materii" (ID, NumeMaterie)
  • Creeaza o interogare care combina mai multe conditii cu AND
  • Adauga campuri extra (Telefon, Email, Adresa)
  • Creeaza un formular pentru introducerea datelor
