Ce este SQL?

Ce cuvant SQL folosesti pentru a specifica ce coloane vrei?

Recapitulare: Operatori in WHERE

Operatorii de comparatie iti permit sa filtrezi datele exact cum vrei: = (egal), > (mai mare), < (mai mic), >= (mai mare sau egal), <= (mai mic sau egal), <> sau != (diferit de).

Exemplu: SELECT * FROM Elevi WHERE Clasa <> '8A' - returneaza toti elevii care NU sunt in clasa 8A.

Ce operator folosesti pentru a gasi elevii care NU sunt din clasa 8A?
A = 8A
B <> 8A
C > 8A
üí° Operatorul pentru "diferit de" este <> sau !=.
