Conceptele din spatele a ceea ce ai facut

Ce este un Document?
Un document este un fisier care contine text (si imagini, tabele etc.). E ca o foaie de hartie, dar digitala - o poti modifica oricand si o poti trimite oriunde.
De ce conteaza? Toate temele, proiectele, si chiar cartile sunt documente. Cand stii sa creezi documente, poti comunica ideile tale frumos si clar.
Microsoft Word
Word este un program pentru crearea documentelor. Este cel mai folosit din lume pentru scris texte. Alternativele includ Google Docs (online) sau LibreOffice Writer (gratuit).

Cum faci asta (pas cu pas):

  • Deschide Word
    Apasa tasta ‚äû Win, scrie "Word", apoi apasa Enter
  • Alege "Document gol"
    Cand se deschide Word, dai click pe "Blank document" sau "Document nou"
  • Scrie textul tau
    Pur si simplu incepe sa tastezi. Cursorul (linia verticala care clipeste) iti arata unde apare textul.
  • Salveaza documentul
    Apasa Ctrl + S (sau File ‚Üí Save)
    Alege locatia (Desktop), scrie numele, apoi click "Save"
üí° Sfat important
Salveaza des! Apasa Ctrl + S la fiecare cateva minute. Daca se inchide accidental computerul, nu pierzi munca.
