Ce este un tabel si de ce il folosim?

Un tabel este un mod de a organiza informatiile in randuri si coloane, ca o grila.

Folosim tabele cand avem date care trebuie comparate sau organizate, de exemplu:

  • Orarul scolii (zile si ore)
  • Lista de cumparaturi (produs si pret)
  • Notele la materii (materie si nota)

Cum inserezi un tabel:

  1. Mergi la tab-ul Insert (Inserare) din panglica
  2. Apasa pe butonul Table (Tabel)
  3. Selecteaza cate coloane si randuri vrei
Din ce tab inserezi un tabel?
A Home (Pornire)
B Insert (Inserare)
C View (Vizualizare)
üí° Insert inseamna a insera/adauga ceva nou in document - de aici adaugi tabele, imagini, etc.
2

Dimensiunile tabelului: Coloane si Randuri

Cand creezi un tabel, trebuie sa specifici dimensiunile lui:

  • Coloane = liniile verticale (de sus in jos)
  • Randuri = liniile orizontale (de la stanga la dreapta)

Cand spunem un tabel "3x4", inseamna:

  • 3 coloane (primul numar)
  • 4 randuri (al doilea numar)

Exercitiu practic:

  1. Deschide Word
  2. Mergi la Insert > Table
  3. Creeaza un tabel 3x4 (3 coloane, 4 randuri)
  4. Completeaza-l cu orarul tau de luni!
Un tabel "3x4" are:
A 3 coloane si 4 randuri
B 4 coloane si 3 randuri
C 12 tabele
üí° Primul numar = coloane (pe orizontala), al doilea = randuri (pe verticala).
3

Personalizarea tabelului

Dupa ce ai creat un tabel, poti sa-l faci mai frumos:

  • Table Design - schimba culoarea si stilul tabelului
  • Layout - adauga/sterge randuri sau coloane
  • Borduri - schimba grosimea si culoarea liniilor

Sfaturi utile:

  • Primul rand se foloseste de obicei pentru titluri (header)
  • Foloseste culori alternate pe randuri pentru citire usoara
  • Aliniaza textul in celule (stanga, centru, dreapta)
Ce poti face pentru a face tabelul mai frumos?
A Poti schimba culoarea celulelor si a chenarelor
B Trebuie sa-l stergi si sa faci altul
C Tabelele nu pot fi modificate
üí° Dupa ce creezi un tabel, poti sa-i schimbi aspectul din Table Design!
4

Adaugarea si stergerea randurilor/coloanelor

Dupa ce ai creat tabelul, poti sa-l modifici adaugand sau stergand randuri si coloane.

Pentru a adauga un rand:

  1. Click dreapta pe un rand din tabel
  2. Alege Insert (Inserare)
  3. Selecteaza Insert Rows Above (deasupra) sau Insert Rows Below (dedesubt)

Pentru a adauga o coloana:

  1. Click dreapta pe o coloana
  2. Alege Insert > Insert Columns Left/Right

Pentru a sterge:

  • Click dreapta > Delete Rows (sterge rand) sau Delete Columns (sterge coloana)
Cum adaugi un rand nou in tabel?
A Stergi tabelul si faci altul mai mare
B Click dreapta in tabel > Insert > Insert Rows Below
C Apesi tasta Enter de multe ori
üí° Click dreapta in tabel deschide un meniu cu optiuni pentru adaugare randuri si coloane.
5

Navigarea in tabel - Celule si Tab

O celula este o casuta din tabel - locul unde un rand se intalneste cu o coloana.

Cum te misti prin tabel:

  • Tab - treci la celula urmatoare (dreapta)
  • Shift + Tab - treci la celula anterioara (stanga)
  • Sagetile - te misti sus/jos/stanga/dreapta
  • Click - mergi direct in celula dorita

Sfat important: Daca apesi Tab in ultima celula a tabelului, se creeaza automat un rand nou!

Exercitiu practic:

  1. Creeaza un tabel 3x3
  2. Scrie ceva in prima celula
  3. Apasa Tab pentru a trece la urmatoarea
  4. Continua pana ajungi la ultima celula
Ce este o celula in tabel?
A Intregul tabel
B O casuta din tabel, la intersectia unui rand cu o coloana
C Primul rand din tabel
üí° Celula este casuta individuala unde scrii - fiecare rand si coloana se intersecteaza intr-o celula.
6

Unirea celulelor (Merge Cells)

Uneori vrei sa combini mai multe celule intr-una singura. De exemplu, pentru un titlu care sa se intinda pe mai multe coloane.

Cum unesti celule:

  1. Selecteaza celulele pe care vrei sa le unesti (click si trage)
  2. Click dreapta si alege Merge Cells (Uneste celule)
  3. Sau: tab-ul Layout > butonul Merge Cells

Exemplu practic: Pentru un orar, poti uni celulele din primul rand pentru a scrie "ORARUL MEU" centrat pe toata latimea.

Atentie:

Daca celulele contin text, doar textul din prima celula va ramane dupa unire!

Cum unesti doua celule intr-una singura?
A Le stergi pe amandoua
B Selectezi celulele si folosesti Merge Cells
C Nu se poate uni celule
üí° Merge Cells inseamna a uni/combina celulele selectate intr-una singura.
7

Proiect: Creeaza-ti orarul scolii!

Acum ca stii sa lucrezi cu tabele, hai sa facem un proiect practic!

PROIECT: Orarul meu

Creeaza un tabel cu orarul tau pentru o zi sau pentru toata saptamana!

Pasi:

  1. Creeaza un tabel 6x7 (6 coloane pentru ore, 7 randuri pentru zile + header)
  2. Primul rand: Luni, Marti, Miercuri, etc.
  3. Prima coloana: orele (8:00, 9:00, etc.)
  4. Completeaza cu materiile tale
  5. Coloreaza diferit fiecare materie!

Sfaturi:

  • Foloseste culori diferite pentru materii diferite
  • Fa textul din header Bold
  • Centreaza textul in celule pentru aspect frumos
Ce proiect poti face cu tabele?
A Doar liste de cumparaturi
B Orar scolar, lista cu note, inventar - orice informatii organizate
C Tabele nu au nicio utilitate
üí° Tabelele sunt foarte versatile - le poti folosi pentru orice tip de informatii care trebuie organizate!
