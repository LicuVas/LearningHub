Software vs Hardware

🎯

GOAL - Obiectivul lectiei

Ce vei invata astazi

"Vreau sa inteleg diferenta dintre partile pe care le ating si programele pe care le folosesc!"

La finalul acestei lectii vei putea:

  • Sa explici diferenta dintre hardware si software
  • Sa identifici sistemul de operare si rolul lui
  • Sa dai exemple de aplicatii si sa le clasifici
💡

TRY - Exploreaza

Hai sa intelegem diferenta!

Stii deja ca hardware-ul sunt piesele pe care le poti atinge. Dar ce pune calculatorul in miscare? Ce il face sa "gandeasca"?

🖥

HARDWARE

"Corpul" calculatorului

  • Poti vedea si atinge
  • Piese fizice, solide
  • Se pot strica sau inlocui
  • Exemple: monitor, tastatura, procesor
💾

SOFTWARE

"Sufletul" calculatorului

  • Nu poti atinge
  • Programe si instructiuni
  • Se poate instala sau sterge
  • Exemple: Windows, jocuri, browser
Software = programele care "dau viata" calculatorului. Fara software, hardware-ul e doar o cutie cu piese care nu face nimic!

💻 Sistemul de operare (OS)

Sistemul de operare este cel mai important software. E ca un "sef" care coordoneaza totul:

Sistemul de operare:
  • Porneste calculatorul si il pregateste pentru lucru
  • Permite ca mouse-ul si tastatura sa functioneze
  • Afiseaza desktop-ul, ferestrele, iconitele
  • Lasa alte programe sa ruleze

Exemple de sisteme de operare:

💻 Windows 🍎 macOS 🐧 Linux 📱 Android 📱 iOS

LEARN - Retine

Informatiile importante

✎ De retinut:

  • Hardware = parti fizice (le poti atinge)
  • Software = programe (nu le poti atinge)
  • Sistemul de operare = programul principal care coordoneaza totul (Windows, macOS)
  • Aplicatii = programe pentru diverse activitati (Word, Chrome, Minecraft)
  • Hardware-ul fara software e inutil; software-ul fara hardware nu poate rula
✅
