Cum creezi si partajezi un document:

  1. Deschide docs.google.com
  2. Apasa "+ Blank" pentru document nou
  3. Da-i un nume (click pe "Document fara titlu")
  4. Apasa butonul albastru "Distribuie" din dreapta sus
  5. Adauga adresele de email ale colegilor
  6. Alege nivelul de acces (Vizualizare/Comentare/Editare)
  7. Apasa "Trimite"!

Ce este Google Docs?

Google Docs este un procesor de text online (ca Microsoft Word, dar in browser). Documentele se salveaza automat in cloud (pe internet) si pot fi accesate de oriunde!
