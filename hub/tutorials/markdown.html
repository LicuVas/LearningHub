Markdown

Limbaj simplu de formatare text pentru documentatie, README si notite

Ce este Markdown?

Markdown este un limbaj de formatare text simplu, creat pentru a fi usor de citit si scris. Se foloseste pentru:

  • Fisiere README pe GitHub
  • Documentatie tehnica
  • Notite si wiki-uri
  • Bloguri si site-uri statice

Titluri (Headings)

Markdown:
# Titlu Mare (H1) ## Subtitlu (H2) ### Sectiune (H3)
Rezultat:

Titlu Mare (H1)

Subtitlu (H2)

Sectiune (H3)

Formatare Text

Markdown:
**text bold** *text italic* ~~text taiat~~ `cod inline`
Rezultat:

text bold

text italic

text taiat

cod inline

Liste

Markdown:
- Element 1 - Element 2 - Sub-element 1. Primul 2. Al doilea 3. Al treilea
Rezultat:
  • Element 1
  • Element 2
    • Sub-element
  1. Primul
  2. Al doilea
  3. Al treilea

Link-uri si Imagini

Markdown:
[Text link](https://example.com) ![Alt text](imagine.png)
Rezultat:

Text link

Poate fi pe mai multe linii.
